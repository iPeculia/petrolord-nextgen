# Material Balance Help System

This folder contains the source code and content for the Material Balance Help & Training system.

## Components
- **HelpGuide.jsx**: The main container for the Help tab.
- **TrainingDashboard.jsx**: The main container for the Training tab.
- **Guides**: Located in `src/components/materialBalance/guides/`.
- **Training**: Located in `src/components/materialBalance/training/`.

## Data
The content for the help system is centralized in `src/data/materialBalanceHelpContent.js`.

## Integration
The help system is integrated into the main `MaterialBalancePro.jsx` page via the `HelpProvider` context.

## Support
For technical issues with the help system, please contact the development team.
