# Quick Start Guide: Permian Basin Sample Data

Follow these steps to immediately start exploring the Decline Curve Analysis module.

## 1. Load the Data
1.  Navigate to the **Decline Curve Analysis** module.
2.  On the Dashboard, click the blue **Load Permian Samples** button.
3.  Wait for the "Project Loaded" confirmation toast.

## 2. Explore Production
1.  Click the **Decline Analysis** tab at the top.
2.  Select **Midland Wolfcamp A 1H** from the well list on the left.
3.  View the historical production graph. Toggle the scale between **Log** and **Linear** using the chart controls.

## 3. Fit a Model
1.  Stay on the **Decline Analysis** tab.
2.  Scroll down to the **Model Parameters** panel.
3.  Observe the pre-fitted values ($b$-factor, $D_i$).
4.  *Optional*: Adjust the parameters manually if "Manual Mode" is enabled.

## 4. Compare Scenarios
1.  Open the **Scenarios** panel on the right sidebar.
2.  The **Base Case** is selected by default.
3.  Click **High Price Environment**.
4.  Navigate to **Forecast Results** to see how the "Economic Limit" and "Remaining Reserves" have changed.

## 5. Analyze Type Curves
1.  Switch the main tab to **Type Curves**.
2.  Select **Midland Wolfcamp A Type Curve** from the dropdown.
3.  Observe how the loaded wells cluster around the type curve.

## 6. View Economics
1.  Switch to the **Adv. Analytics** tab.
2.  View the **Scenario NPV Sensitivity** chart to see the financial impact of different price decks.

## 7. Reset
1.  Return to the **Dashboard** tab.
2.  Click **Clear Data** to remove the sample project and start fresh.
