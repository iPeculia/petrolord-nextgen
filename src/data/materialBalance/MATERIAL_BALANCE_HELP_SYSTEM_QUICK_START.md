# Help System Quick Start Guide

Get the answers you need, right when you need them.

## 1. Accessing Help
You can access help in three ways:
1.  **Help Tab**: Click "Help" in the main navigation bar for the full documentation library.
2.  **Assistant Sidebar**: Click the `?` icon in the top-right corner for quick search and recent topics.
3.  **Context Tooltips**: Hover over any UI element (charts, buttons) to see a popup explanation.

## 2. Searching
1.  **Global Search**: Type in the search bar within the Help Sidebar or Help Tab.
2.  **Smart Results**: Results are categorized by type (Guide, FAQ, Glossary). Click any result to view.

## 3. Using Tutorials
1.  Go to the **Training** tab.
2.  Click "Start Module" on any card (e.g., "Fundamentals of MB").
3.  Follow the on-screen prompts to step through the workflow.

## 4. Watching Videos
1.  Go to **Training > Videos**.
2.  Click on a video thumbnail to play.
3.  Videos cover specific tasks like "Importing Data" or "Creating Scenarios".

## 5. Keyboard Shortcuts
*   Press `?` anywhere to open the Help Assistant.
*   Press `Ctrl + F` to focus the search bar.
*   See the **Keyboard Shortcuts** guide in the Help Tab for a full list.

## 6. Customizing Your Experience
1.  Open the Help Sidebar.
2.  Click **Settings** at the bottom.
3.  Toggle **Contextual Tooltips** on/off based on your preference.
4.  Manage **Notifications** for new training content.

## 7. Getting Support
*   **Feedback**: Use the "Send Feedback" button in the sidebar to report bugs or suggestions.
*   **Contact Support**: Use the "Contact Support" button for critical issues requiring technical assistance.
